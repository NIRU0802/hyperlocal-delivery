## ğŸš¦ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/NIRU0802/hyperlocal-delivery.git

# Navigate to project directory
cd hyperlocal-delivery

# Install dependencies

# Start development server
npm run dev
```
