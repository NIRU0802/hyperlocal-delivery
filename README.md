# Hyperlocal Delivery

A Next.js hyperlocal delivery application that enables users to order food and groceries from nearby restaurants and stores.

## Features

- Browse restaurants and food options
- Quickmart for grocery delivery
- Instamart for instant deliveries
- Cart management
- Order tracking with live map
- Delivery partner integration
- Rider dashboard
- User authentication

## Tech Stack

- Next.js
- React
- Tailwind CSS
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
